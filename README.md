GitLab Time Tracker - lightweight browser client for logging and viewing GitLab time entries.

A minimal SPA that authenticates with a GitLab token, shows projects/issues, lets you add/delete time logs, and displays a month calendar - all data cached locally to avoid repeated API calls.
